## 요약

AE gain LUT 탐색 함수가 경계 idx 에서 off-by-one 반환하던 것을 명시 경계 분기로 교정.

## 상세 설명

AE 탐색 함수의 인덱스 반환 로직이 경계 조건에서 잘못된 값을 반환해 호출자가 잘못된 LUT 엔트리(경계 밖)를 포인터 연산으로 접근할 가능성을 줄임.

## 취약 코드 (OLD)

```c
result = a4 - v7;
v16 = a3 - v7;
v17 = a4 - v7 + 1;

```

## 수정된 코드 (NEW)

## 공격 시나리오

AE 설정을 통해 정상 범위를 벗어난 v7/a3/a4 가 들어오면 OLD 는 a4-v7 을 무조건 반환해 호출자가 LUT[v7+idx] 를 역참조 시 OOB read 소량 가능.

## 수정 설명

v17<=v16 이면 v17-1 로 즉시 반환, 이후 근접 비교도 unsigned 64-bit 로 통일해 경계 계산을 안전화.
