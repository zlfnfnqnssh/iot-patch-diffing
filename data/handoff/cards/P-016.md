## 요약

sprintf 포맷 문자열에 %u 있으나 대응 인자 미공급으로 스택 잔존값 노출. 수정본에서 인자 매칭 완료.

## 상세 설명

OLD 코드는 sprintf 포맷 지정자에 대응 인자를 일부 생략해 호출 시 레지스터/스택 잔존 값이 포맷팅 결과에 삽입됐다. root 권한 proc read에서만 접근 가능해 익스플로잇 표면은 제한적이나 커널 정보 누수 가능성 존재.

## 취약 코드 (OLD)

```c
sprintf((char *)(a1 + v19), "--Send Elem Nummber : %u\n");
sprintf((char *)(a1 + v20), "--Recv Elem Nummber : %u\n");
```

## 수정된 코드 (NEW)

```c
sprintf((char *)(a1 + v18), "--Send Elem Nummber : %u\n", v25);
sprintf((char *)(a1 + v19), "--Recv Elem Nummber : %u\n", v26);
```
